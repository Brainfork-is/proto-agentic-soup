# LangChain.js Built-in Tools Reference

## Overview

This document catalogs the available built-in tools in LangChain.js that we can leverage for our real-world agent ecosystem. Tools are utilities designed to be called by models, with inputs generated by AI and outputs fed back to AI systems.

## Core Tool Categories

### 1. Web & Search Tools

#### Web Browser
```typescript
import { WebBrowser } from "@langchain/community/tools/webbrowser";
```
- **Purpose**: Visit websites and extract content
- **Use Cases**: Real-time web research, content extraction
- **Real-world value**: Agents can research current information

#### Search Engines

## Tool Usage Patterns

### Basic Tool Usage
```typescript
const tool = new ToolName(configuration);
const result = await tool.invoke(input);
```

### Tool with Agent
```typescript
import { createReactAgent } from "@langchain/langgraph/prebuilt";

const agent = createReactAgent({
  llm: model,
  tools: [tool1, tool2, tool3],
});
```

### Custom Tool Creation
```typescript
import { DynamicTool } from "@langchain/core/tools";

const customTool = new DynamicTool({
  name: "custom_tool",
  description: "Does something specific",
  func: async (input: string) => {
    // Implementation
    return result;
  },
});
```
